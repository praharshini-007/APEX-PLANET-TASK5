Medical Diagnosis Bot

An AI-based medical diagnosis system using CNN and health image data.

Overview

This bot predicts possible medical conditions using image classification with a CNN trained on public medical datasets. The interface is clean, and the model is deployed on the web with user input support.

Features

  • Convolutional Neural Network (CNN) for medical image classification
  • Web interface for user diagnosis
  • Trained on real health datasets
  • Real-time output display

Technologies Used

Python, TensorFlow, CNN, HTML, CSS, JavaScript
