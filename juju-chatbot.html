JuJu Chatbot

An intelligent NLP chatbot built using NLTK and deployed on a responsive web interface.

Overview

JuJu is a rule-based chatbot developed using Natural Language Processing (NLP) techniques. It responds to user queries with contextual answers and features a polished web interface made with HTML, CSS, and JavaScript.

Features

  • Simple and fast conversational flow
  • Built with NLTK in Python
  • Custom trained intents
  • Responsive web-based UI

Technologies Used

Python, NLTK, HTML, CSS, JavaScript
